autofill

#repo
Package using selenium to fill form and submit (with Google Captcha passing)

Env

  • CAPTCHA_API_KEY - Captcha api key (get from 2captcha.com)
  • PAGE_URL - url that contains form need to submit
  • GOOGLE_KEY - Google captcha key (get from page)

Endpoint

POST /process

Input format (JSON)

Supported types

  • textbox: for selection, textbox, radio button
  • checkbox: checkbox
  • Will support more …
